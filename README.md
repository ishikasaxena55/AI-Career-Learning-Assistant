# AI Career & Learning Assistant 🚀

## 📌 Project Description
An AI-powered resume analyzer that uses Groq API to evaluate resumes, calculate ATS score, detect missing skills, and provide personalized career guidance.

## ✨ Features
- Resume Upload (PDF)
- ATS Score Calculation
- Skill Gap Analysis
- AI Career Suggestions using Groq API
- Streamlit Web App
- Live Deployment on Render

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API
- PyPDF2
- Pandas
